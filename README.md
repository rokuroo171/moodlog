# moodlog 🌙

Ever forgot how you felt last tuesday? yeah. moodlog fixes that.

Pick your mood, write a note, done. it remembers so you don't have to.

## What it does...

- 6 moods — happy, sad, angry, anxious, calm, tired
- optional note for each entry
- saves everything to localStorage (no backend, no account)
- shows your last 7 logs

## Stack

Just HTML, CSS, and JS.
